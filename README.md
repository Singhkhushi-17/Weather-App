# Weather App

A responsive weather application that provides real-time weather updates using a weather API.

## Features
- Search weather by city name
- Displays temperature
- Shows humidity and wind speed
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
